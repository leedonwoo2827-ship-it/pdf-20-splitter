# PDF 20장씩 자르기 (pdf-20-splitter)

교정 업무용 PDF 분할 도구. 시작 페이지를 지정하면 그 지점부터 **20페이지씩** 자동으로 나눠 저장합니다.

- 초교 예: 시작 페이지 `20` → `20~39`, `40~59`, `60~79` …
- 재교 예: 시작 페이지 `30` → `30~49`, `50~69`, `70~89` …

---

## 다운로드

1. 이 레포지토리의 **[Releases](../../releases)** 페이지로 이동
2. 최신 버전에서 `PDF20장자르기.exe` 클릭 → 다운로드
3. 원하는 폴더(예: 바탕화면)에 저장

> Windows 전용입니다. 별도 설치 없이 바로 실행됩니다.

## 사용법

1. 다운로드한 `PDF20장자르기.exe`를 더블클릭합니다.
2. **PDF 파일** 「찾기…」 버튼으로 원본 PDF 선택
3. **시작 페이지**에 숫자 입력 (초교 20, 재교 30 등)
4. 필요하면 **출력 폴더** 「변경…」 버튼으로 저장 위치 지정 (기본값: 원본 폴더 안의 `split` 폴더)
5. **분할 실행** 클릭

완료되면 자동으로 출력 폴더가 열리고, 다음과 같이 파일이 생성됩니다:

```
원본이름_p020-039.pdf
원본이름_p040-059.pdf
원본이름_p060-079.pdf
...
```

## SmartScreen 경고가 뜨면

서명되지 않은 실행 파일이라 Windows에서 최초 1회 경고가 뜰 수 있습니다.

1. 「추가 정보」 클릭
2. 「실행」 버튼 클릭

## 제한 사항

- 암호가 걸린 PDF는 열 수 없습니다.
- 시작 페이지가 전체 페이지 수를 넘으면 오류 메시지가 표시됩니다.

## 개발자용 (로컬 실행)

```bash
pip install -r requirements.txt
python src/splitter.py
```

## 개발자용 (로컬 빌드)

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name "PDF20장자르기" src/splitter.py
# dist/PDF20장자르기.exe 생성
```
